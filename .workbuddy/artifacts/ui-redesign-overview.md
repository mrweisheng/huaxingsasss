# 前端UI重构 — 设计变更概览

## 品牌方向

**华星资源合同管理系统** — 针对粤港两地牌/车辆买卖企业级管理场景

- **主色**：深海军蓝 `#1e3a5f` — 传递信赖、权威、专业
- **点缀色**：金色 `#c9952b` — 高端商务质感
- **字体**：`Noto Serif SC` 标题（中式专业感）+ `JetBrains Mono` 等宽数字
- **调性**：Premium · Professional · Authoritative

## 核心变更

### 1. 全局设计系统 (`index.css`)
- 完整 CSS 变量体系（品牌色/语义色/中性色/阴影/圆角/字体）
- Ant Design 全局主题覆写（按钮/输入框/表格/标签/菜单）
- 通用组件类（`page-topbar` / `page-title-wrap` / `page-title-icon` / `card-premium`）

### 2. 布局 (`Layout.tsx`)
- 侧边栏品牌化：华星Logo（金底黑字）+ 中英文标题 + 金色分隔线
- 顶部栏毛玻璃效果（`backdrop-filter: blur(12px)`）
- 底部用户信息区（头像+角色）
- 可折叠侧栏

### 3. 登录页 (`Login.tsx`)
- 左半幅：深色渐变品牌展示区 + 金色Logo + 装饰光晕 + 产品特色点
- 右半幅：干净清爽的登录表单

### 4. 页面级更新
- **合同/客户列表**：统一的 `page-topbar` 风格 + 品牌色图标容器
- **合同卡片**：更精致的阴影、更好的间距比例
- **智能问答**：金色assistant气泡、用户消息渐变蓝色气泡、空状态品牌引导区
- **付款管理**：表格圆角、状态标签背景色增强
