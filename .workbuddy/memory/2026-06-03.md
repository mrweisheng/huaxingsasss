# 2026-06-03

## 客户管理页面卡片重设计

重新设计了客户列表页的客户卡片，嵌入关联合同信息展示。

### 后端改动
- **contracts.py**: 新增 `customer_ids` 查询参数（逗号分隔），支持批量按客户ID查合同
- **contract_service.py**: `get_contracts()` 新增 `customer_ids: Optional[List[int]]` 参数，使用 `Contract.customer_id.in_(customer_ids)` 批量过滤

### 前端改动
- **contract.ts**: `getList` 参数类型新增 `customer_ids?: string`
- **CustomerList.tsx**: 完全重写
  - 客户列表加载后，提取所有 customer.id → 批量调用 `contractApi.getList({ customer_ids })` → 按 customer_id 分组
  - 卡片内新增"关联合同"区域，最多显示2份合同，超过显示"+N 份合同"
  - 每份合同显示：编号、业务类型徽章、状态圆点、金额、付款进度条
  - 点击合同行 → `navigate(/contracts/:id)`（stopPropagation）
  - 点击整张卡片 → `/customers/:id`（保持原有行为）
  - 无合同时显示"暂无关联合同"占位
  - 加载中显示骨架屏
  - 联系人信息改为紧凑一行显示
- **CustomerList.css**: 新增合同行、进度条、业务类型徽章、状态圆点、骨架屏等样式；卡片最小宽度调整为360px
