# 凭证图片不显示：`_ensure_file_in_receipt_dir` 缺少 AGENT_FILE_DIR 搜索路径

## 问题

Agent 录入收款后，收付管理/合同台账看不到凭证原图。日志：
```
凭证文件复制跳过: 临时文件不存在 file_id=fa08f0a8-..., user=1
```

## 根因

`tools.py:721-724` 的 `_ensure_file_in_receipt_dir` 只搜索 `TEMP_UPLOAD_DIR` 两个子目录，而文件实际存储在 `AGENT_FILE_DIR/{user_id}/`。`resolve_file_path`（`file_utils.py:139`）已加 `AGENT_FILE_DIR` 优先搜索，但 `_ensure_file_in_receipt_dir` 遗漏了。

## 修复

### 文件：`backend/app/ai/tools.py`

**修改 `_ensure_file_in_receipt_dir` 的 candidates 搜索列表**（约 line 721-724）：

将：
```python
for base_dir in [
    os.path.join(settings.TEMP_UPLOAD_DIR, str(self.user.id)),
    settings.TEMP_UPLOAD_DIR,
]:
```

改为：
```python
for base_dir in [
    os.path.join(settings.AGENT_FILE_DIR, str(self.user.id)),  # 新：持久化目录优先
    os.path.join(settings.TEMP_UPLOAD_DIR, str(self.user.id)),
    settings.TEMP_UPLOAD_DIR,
]:
```

与 `file_utils.py:resolve_file_path` 的搜索顺序对齐。

## 验证

1. 启动后端：`cd backend && uv run uvicorn app.main:app --reload`
2. 通过 Agent 上传凭证图片 → 录入收入
3. 检查日志不再出现"凭证文件复制跳过"
4. 收付管理/合同台账页面可查看凭证原图
