业务色彩基调 · 车辆 vs 两地牌

定义两个核心业务的完整用色系统（主色 / 浅底 / 深字 / hero 渐变），让用户在任何页面一眼分辨业务类型。 本调研基于 Linear / Stripe / Mercury / Notion 的业务色实践 + 项目已有 token， 给出三个方向。选定后写入 CLAUDE.md，后续所有页面统一使用。

⚠ 当前色系冲突（必须解决）
现版「两地牌业务」使用 #0d9488， 与「全额结清」状态色 #0d9488 完全相同。 一份「未结清的两地牌合同」和一份「已结清的车辆合同」放在同一屏时，用户无法靠颜色区分语义。 必须让两地牌业务避开 teal 色相，把 teal 留给状态语义。
A

方向 A · 钢蓝 × 朱砂（实物派）

汽车工业漆 · 中式印章红 · 高识别度

语义自洽度最高。车辆 = 汽车厂的金属漆面钢蓝（保时捷/奔驰深色系），两地牌 = 朱砂红（车牌红边/通关印章/中港旗帜）。 色相距离 200°，扫视区分度最强。朱砂红与项目金色形成「红金」中式财务美学，气质与「华星」品牌契合。 缺点：朱砂红视觉重量略高于钢蓝，在列表中可能显得"两地牌"比"车辆"更显眼。

在现有 #667eea 紫上加深稳定 · 两地牌切换到朱砂避冲突

方向对比 · 选型建议

方向 车辆色 两地牌色 语义 状态色冲突 品牌契合 迁移成本
A · 钢蓝×朱砂 #2d5b8a 钢蓝 #b8423b 朱砂 汽车工业 + 印章/车牌红 ✅ 完全无冲突 ★★★★★ 中式财务红金 中（两个色都改）
B · 燃油橙×港湾青 #d96b2d 燃油橙 #2d7a8c 港湾青 速度 + 海港跨境 ❌ 都撞状态色 ★★★ —
C · 靛紫×朱砂 #4f5db0 深靛紫 #b8423b 朱砂 科技感 + 印章/车牌红 ✅ 无冲突 ★★★★ 不如钢蓝实 低（只改两地牌）
我的推荐：方向 A · 钢蓝 × 朱砂

理由：
1. 语义最强：钢蓝直接呼应汽车工业涂装（保时捷海军蓝/奔驰深空蓝/宾利皇家蓝都是这个家族），朱砂是中港车牌的红边色 + 通关印章色，两个业务都有"实物原型"做支撑，不是凭空指派
2. 状态色零冲突：钢蓝与品牌深蓝同家族但更冷（不抢戏），朱砂与暖橙 due 有 18° 色相差（可分辨）
3. 红金组合契合「华星」品牌：朱砂红 + 已有金色 = 中式财务美学，气质统一
4. 色相距离 218°：两业务在色环上几乎对立，扫视区分度最强
